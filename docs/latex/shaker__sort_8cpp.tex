{\footnotesize\ttfamily void shaker\+\_\+sort (\begin{DoxyParamCaption}\item[{int}]{arr\mbox{[}$\,$\mbox{]},  }\item[{const unsigned int}]{size }\end{DoxyParamCaption})}

Implement the Shaker sort that is a modified of the bubble sort. 

Instead of shift biggest items to to the end of the array for each loop, it shifts the biggest item to end for the first pass.

And for the second pass, it shifts the smallest item to the start of the array. And repeats the process with both directions until the array is sorted
